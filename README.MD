# webkit-cssui-ios7-bubble
> Webkit css ui module for ios7 style bubble.

## usage:
```html
<div class="clfix chat-ui">
  <h1>IOS7-bubble</h1>
  <div class="tl">
    <div class="ios7-bubble" data-bubble-type="left">
      在吗？
    </div>
  </div>
  <div class="tr">
    <div class="ios7-bubble" data-bubble-type="right">
      嗯？
    </div>
  </div>
  <div class="tl">
    <div class="ios7-bubble" data-bubble-type="left">
      你好，我是新来的。
    </div>
  </div>
  <div class="tr">
    <div class="ios7-bubble" data-bubble-type="right">
      呵呵哒
    </div>
  </div>
</div>
```
